# GuessTheFood

GuessTheFood est une application web de jeu autour de la cuisine.

Le but est de deviner les ingredients d'un plat et de reconnaitre des recettes de cuisine.

## Objectif

L'application permettra de :

- proposer un plat ou une recette ;
- faire deviner les ingredients ;
- verifier les reponses du joueur ;
- calculer un score.

## Technique

Le projet contient actuellement une API backend.

- .NET 9
- ASP.NET Core
- Entity Framework Core
- PostgreSQL
- Docker Compose
- Swagger

La base du projet contient deja des entites pour les utilisateurs, les roles, les plats et les ingredients.
